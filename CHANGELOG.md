## 1.0.5

- Switch iOS Cast integration to the Apple Silicon simulator-compatible Google Cast XCFramework pod by default.
- Add an opt-in legacy iOS pod fallback for older iOS targets or device-only builds.
- Remove outdated simulator architecture restrictions from the iOS podspec.
- Raise the default iOS Cast deployment target to 15.0 to match the current Google Cast SDK support policy.

## 1.0.4

- Remove deprecated Android v1 embedding registration.
- Modernize Dart and example app constraints for newer Flutter versions.
- Keep Chromecast/AirPlay functionality intact while updating build files.
